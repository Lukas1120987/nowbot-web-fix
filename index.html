Essential Commands

Easy-to-use commands that give you complete control over your server management.

/ticket setup
Configure the ticket system with custom categories and settings.
/ticket close
Close the current ticket and generate a transcript.
/mod warn
Issue a warning to a user with custom reason.
/mod clear
Bulk delete messages from a channel.
/poll create
Create engaging polls with multiple options.
/userinfo
Display detailed information about a user.
/serverinfo
Show comprehensive server statistics.
/help
View all available commands and their usage.
